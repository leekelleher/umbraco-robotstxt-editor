Robots.txt Editor

Welcome to the Umbraco Robots.Txt editor dashboard!

Below you can enter the files and folders you wish to be ingnored by web-crawlers and search engines.

To add a comment to the file, start the lnie with the # character, for example:

# Welcome to my robots file

To add a file or folder, type the word 'Disallow' followed by a semi-colon, for example:

Begin and end the filepath with a forward slash

Disallow: /bin/

Add User-Agent to prevent a particular web crawler by typing the phrase 'User-Agent' and declare the name of the particular bot you wish to disallow.

If you wish to prevent all bots and crawlers, just add a star after the User-Agent declaration.

Followed this with 'Disallow' and a forward slash to include the entire site, for example

User-Agent: *

Disallow: /
